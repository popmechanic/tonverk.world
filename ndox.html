01

Clipboard

Copy, paste, and clear operations for tracks, trigs, patterns, presets, and pages. Includes undo/redo.

Track Sequence — in Grid Recording mode

FUNC + RECORD
Copy the active track's sequence
FUNC + STOP
Paste the copied track's sequence to the active track
FUNC + PLAY
Clear the active track's sequence

Trig — in Grid Recording and Live Recording mode

TRIG + RECORD
Copy the trig with its parameter locks
TRIG + STOP
Paste the copied trig
TRIG + PLAY
Clear all parameter locks on the trig

Pattern

FUNC + RECORD
Copy the active pattern
FUNC + STOP
Paste the copied pattern to the active pattern
FUNC + PLAY
Clear the active pattern

Preset

TRK + RECORD
Copy the selected track's preset
TRK + STOP
Paste the copied preset to the selected track
TRK + PLAY
Clear the selected track's preset

Sequencer Page — in Grid Recording mode

PAGE + RECORD
Copy the active page
PAGE + STOP
Paste the copied page to the active page
PAGE + PLAY
Clear the active page

Parameter Page

PARAMETER PAGE + RECORD
Copy the selected parameter page
PARAMETER PAGE + STOP
Paste the copied parameter page (select new track first)
PARAMETER PAGE + PLAY
Clear the parameter page and reset to defaults

Preset to Another Track

TRK + TRIG 1-16 + RECORD
Copy a track preset
TRK + TRIG 1-16 + STOP
Paste the preset to the selected track

Multiple Trigs — in Grid Recording mode

Hold trigs + RECORD
Copy multiple trigs
Hold destination TRIG + STOP
Paste the copied sequence of trigs
Hold trigs + PLAY
Clear trig locks on selected trigs

Undo / Redo

FUNC + NO
Undo paste/clear operations on patterns, track sequences, and sequencer pages
FUNC + YES
Redo a previously performed Undo operation
02

Sequencer

Configure tempo, recording modes, playback, and fill patterns for live and step-based sequencing.

Sequencer Settings

FUNC + TEMPO
Tap tempo
LEFT / RIGHT (hold ARROW)
Nudge tempo up/down by 10% temporarily
FUNC + LEFT / RIGHT
Move all trigs on the track a whole step left or right (in Grid Recording)
TRIG (hold) + LEFT / RIGHT
Open the Micro Timing menu and set micro timing for that trig (in Grid Recording)
FUNC + KEYBOARD F#1
Toggle the metronome on/off
FUNC + KEYBOARD G#
Open the Quantize menu
FUNC + KEYBOARD A#1
Open the Scale menu
Hold FUNC + turn DATA ENTRY knob E
Change pattern length in even steps of 16

Sequencer Recording

RECORD
Activate/deactivate Grid Recording mode
RECORD + PLAY
Start Live Recording mode
RECORD + double-press PLAY
Activate/deactivate Quantize Live Recording
STEP EDIT
Enter Step Edit mode (in normal playback)
Hold PAGE + LEFT / RIGHT
Change sequencer page (in Grid Recording)
Hold PAGE + TRIG
Jump to that page (in Grid Recording)
PAGE
Select the active pattern page (if more than 16 steps)
FUNC + TRIG
Add a lock trig (in Grid Recording)

Playback Control

PLAY
Start / pause / resume sequencer playback
STOP
Stop playback
STOP + STOP
Stop with short fade-out of all tracks and send effects

Fill Mode

FILL (Momentary)
Activate Fill for as long as you hold the key
FILL (Latching)
Toggle Fill mode on/off
FUNC + FILL
Open the Fill Edit menu
03

Tracks &
Patterns

Navigate tracks, subtracks, banks, and patterns. Control mutes and transition modes.

Track / Subtrack / Pattern / Bank Select

TRK + TRIG 1-16
Select a track
TRK + SUBTRACKS
Select a subtrack (on a track with Subtracks SRC machine)
PTN + KEYBOARD (bottom row) + TRIG 1-16
Select a bank/pattern
TRIG 1-16 (not in recording mode)
Select a track and trigger its preset (audio tracks)

Pattern Selection

PTN + TRIG 1-16
Select a pattern within the current bank
PTN + SUBTRACKS
Select a bank (then TRIG 1-16 to choose pattern)

Mutes

MUTE
Enter Mute mode
MUTE + TRIG / SUBTRACKS
Quick mute/unmute one or several tracks
MUTE + UP
Enter Global Mute mode
MUTE + DOWN
Enter Pattern Mute mode
MUTE + UP / DOWN (in Mute mode)
Switch between Global and Pattern Mute modes

Transition Mode

FUNC + PTN
Open the Transition mode select menu
Hold FUNC + press PTN repeatedly
Cycle through Transition modes
04

Performance

Perform mode, arpeggiator controls, transpose, and song mode for live sets and arrangements.

Perform Mode

FUNC + TRK
Toggle Perform mode on/off

Arpeggiator

FUNC + ARP
Toggle the current track's arpeggiator on/off
FUNC + LEFT / RIGHT (in ARP menu)
Rotate the arpeggiator sequence

Transpose

+ / -
Transpose the Keyboard up/down in octaves
FUNC + + then use KEYBOARD
Transpose the current track's note trigs up/down to the selected key

Song Mode

FUNC + SONG
Toggle Song mode on/off
SONG + TRIG 1-16
Select a song and enter Song mode
SONG + RIGHT
Open the Song Edit screen
SONG + LEFT (in Song mode)
Loop the currently playing row; press again to stop looping
SONG + UP / DOWN (in Song mode)
Select a specific song row to jump to and play next
STOP + STOP (in Song mode)
Move the playhead to the song's beginning

Song Edit Screen

FUNC + RECORD
Copy the selected row
FUNC + STOP
Paste a previously copied row to the selected row
FUNC + PLAY
Reset the selected row's settings (default BPM, length, mute)
FUNC + DOWN
Add a new row below the selected row (copy of selected)
FUNC + UP
Delete the selected row

Parameter Editing

PARAMETER PAGE + PLAY
Reset all parameters on the selected parameter page to defaults
PARAMETER PAGE + YES

DATA ENTRY knob + NO
Reset the individual parameter to its default value
Press and turn DATA ENTRY knob
Adjust parameter in larger increments
Hold FUNC + turn DATA ENTRY
Snap parameter values to musically relevant values (octaves, center, etc.)
Press and hold PARAMETER PAGE
See the values for all parameters on that page

Preview

FUNC + YES (in Sample browser)
Preview the selected sample
TRIG + YES (in Grid Recording)
Preview a specific trig (including its parameter locks)
KEYBOARD + YES (in Step Edit)
Preview a specific note (including its parameter locks)

CHORD

TEMPO

06

Sampling

Browse, record, trim, and save samples. Manage your project files and recall patterns.

Sample Browser

FUNC + SAMPLING
Open the Sample browser
YES or FUNC + RIGHT
Enter the highlighted directory
NO or FUNC + LEFT
Exit a directory / go up one level
Hold NO
Return to the top of the directory structure
FUNC + NO
Exit the Sample browser
RIGHT
Access the Operations menu for the highlighted item

Sampling (Recorder)

FUNC + YES (in trim screen)
Preview the sample
LEFT (in trim screen)
Toggle sample Loop on/off
UP (in trim screen)
Normalize the sample to 0 dB
FUNC + NO (in trim screen)
Discard the sample

Saving & Reloading

FUNC + SETTINGS
Save the active project
FUNC + KEYBOARD C#
Temporarily memorize (save) the active pattern
FUNC + KEYBOARD D#
Recall (reload) the active pattern from its temporarily saved state
07

System

Naming screens, keyboard secondary functions, rescue mode, routing, and general navigation shortcuts.

Naming Screen

LEFT / RIGHT or LEVEL/DATA
Navigate between characters
UP / DOWN or LEVEL/DATA
Select characters
FUNC + ARROW
Choose a letter (pop-up naming)
FUNC + NO
Erase a character
FUNC + YES
Insert a blank space
SETTINGS
Generate a random name
Hold FUNC
Open the Pop-up Naming menu
FUNC + TRIG 1 / TRIG 2
Switch between character layouts (in pop-up naming)

Keyboard Secondary Functions

KEYBOARD C#1
Recall from previously memorized state of the active pattern
KEYBOARD D#1
Memorize the state of the active pattern
KEYBOARD F#1 (via FUNC)
Toggle the metronome on/off
KEYBOARD G#1 (via FUNC)
Open the Quantize menu
KEYBOARD A#1 (via FUNC)
Open the Scale menu

Rescue Mode

Hold FUNC while powering on
Enter Rescue Mode menu
TRIG 1 (in Rescue Mode)
Reset active project
TRIG 2 (in Rescue Mode)
Recovery OS upgrade
TRIG 3 (in Rescue Mode)
Save diagnostics
TRIG 4 (in Rescue Mode)
Exit (reboot)

Routing

FUNC + MUTE
Open the Routing menu
Hold TRIG keys + press destination TRIG
Route track(s) to a destination in the Routing menu
KEYBOARD A / KEYBOARD B (in Routing)
Select OUT CD and OUT EF destinations respectively

Miscellaneous

FUNC + UP/DOWN (in menus)
Move the cursor one menu page at a time
YES
Confirm, select, enter sub-menus, tick/untick boxes
NO
Cancel, deselect, go back one step, perform Undo action
FUNC + NO
Undo action
FUNC + YES
Redo action
Hold POWER then press YES
Turn off the device
Hold POWER button (unresponsive)
Force power off
FUNC + TRIG (on Naming pop-up)
Switch character layouts
