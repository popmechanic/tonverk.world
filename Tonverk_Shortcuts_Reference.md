# Elektron Tonverk — Complete Shortcuts & Key Combinations Reference

*Extracted from the Tonverk User Manual, OS 1.2.1 (Feb 4, 2025)*

---

## COPY / PASTE / CLEAR

### Track Sequence (all trigs on the track) — in GRID RECORDING mode

| Shortcut | Function |
|----------|----------|
| **[FUNC] + [RECORD]** | Copy the active track's sequence |
| **[FUNC] + [STOP]** | Paste the copied track's sequence to the active track |
| **[FUNC] + [PLAY]** | Clear the active track's sequence |

### Trig — in GRID RECORDING and LIVE RECORDING mode

| Shortcut | Function |
|----------|----------|
| **[TRIG] + [RECORD]** | Copy the trig with its parameter locks |
| **[TRIG] + [STOP]** | Paste the copied trig |
| **[TRIG] + [PLAY]** | Clear all parameter locks on the trig |

### Pattern

| Shortcut | Function |
|----------|----------|
| **[FUNC] + [RECORD]** | Copy the active pattern |
| **[FUNC] + [STOP]** | Paste the copied pattern to the active pattern |
| **[FUNC] + [PLAY]** | Clear the active pattern |

### Preset

| Shortcut | Function |
|----------|----------|
| **[TRK] + [RECORD]** | Copy the selected track's preset |
| **[TRK] + [STOP]** | Paste the copied preset to the selected track |
| **[TRK] + [PLAY]** | Clear the selected track's preset |

### Sequencer Page — in GRID RECORDING mode

| Shortcut | Function |
|----------|----------|
| **[PAGE] + [RECORD]** | Copy the active page |
| **[PAGE] + [STOP]** | Paste the copied page to the active page |
| **[PAGE] + [PLAY]** | Clear the active page |

### Parameter Page

| Shortcut | Function |
|----------|----------|
| **[PARAMETER PAGE] + [RECORD]** | Copy the selected parameter page |
| **Same [PARAMETER PAGE] + [STOP]** | Paste the copied parameter page (select new track first) |
| **[PARAMETER PAGE] + [PLAY]** | Clear the parameter page and reset to defaults |

### Preset to Another Track

| Shortcut | Function |
|----------|----------|
| **[TRK] + [TRIG 1–16] + [RECORD]** | Copy a track preset |
| **then [TRK] + [TRIG 1–16] + [STOP]** | Paste the preset to the selected track |

### Multiple Trigs (in GRID RECORDING mode)

| Shortcut | Function |
|----------|----------|
| **Hold trigs + [RECORD]** | Copy multiple trigs |
| **Hold destination [TRIG] + [STOP]** | Paste the copied sequence of trigs |
| **Hold trigs + [PLAY]** | Clear trig locks on selected trigs |

## UNDO / REDO

| Shortcut | Function |
|----------|----------|
| **[FUNC] + [NO]** | Undo paste/clear operations on patterns, track sequences, and sequencer pages |
| **[FUNC] + [YES]** | Redo a previously performed Undo operation |

## TRACK / SUBTRACK / PATTERN / BANK SELECT

| Shortcut | Function |
|----------|----------|
| **[TRK] + [TRIG 1–16]** | Select a track |
| **[TRK] + [SUBTRACKS]** | Select a subtrack (on a track with Subtracks SRC machine) |
| **[PTN] + [KEYBOARD] (bottom row) + [TRIG 1–16]** | Select a bank/pattern |
| **[TRIG 1–16]** (not in recording mode) | Select a track and trigger its preset (audio tracks) |

## NAMING SCREEN

| Shortcut | Function |
|----------|----------|
| **[LEFT] / [RIGHT]** or *LEVEL/DATA* | Navigate between characters |
| **[UP] / [DOWN]** or *LEVEL/DATA* | Select characters |
| **[FUNC] + [ARROW]** | Choose a letter (pop-up naming) |

| **[SETTINGS]** | Generate a random name |
| **Hold [FUNC]** | Open the Pop-up Naming menu |

## SAVING AND RELOADING

| Shortcut | Function |
|----------|----------|
| **[FUNC] + [SETTINGS]** | Save the active project |
| **[FUNC] + [KEYBOARD C#]** | Temporarily memorize (save) the active pattern |
| **[FUNC] + [KEYBOARD D#]** | Recall (reload) the active pattern from its temporarily saved state |

## SEQUENCER SETTINGS

| Shortcut | Function |
|----------|----------|
| **[FUNC] + [TEMPO]** | Tap tempo |
| **[LEFT] / [RIGHT]** (on main screen, hold [ARROW]) | Nudge tempo up/down by 10% temporarily |
| **[FUNC] + [LEFT] / [RIGHT]** (in GRID RECORDING) | Move all trigs on the track a whole step left or right |
| **[TRIG] (hold) + [LEFT] / [RIGHT]** (in GRID RECORDING) | Open the MICRO TIMING menu and set micro timing for that trig |
| **[FUNC] + [KEYBOARD F#1]** | Toggle the metronome on/off |
| **[FUNC] + [KEYBOARD G#]** | Open the QUANTIZE menu |
| **[FUNC] + [KEYBOARD A#1]** | Open the SCALE menu |
| **Hold [FUNC] + turn *DATA ENTRY* knob E** | Change pattern length in even steps of 16 |

## SEQUENCER RECORDING

| Shortcut | Function |
|----------|----------|
| **[RECORD]** | Activate/deactivate GRID RECORDING mode |
| **[RECORD] + [PLAY]** | Start LIVE RECORDING mode |
| **[RECORD] + double-press [PLAY]** | Activate/deactivate QUANTIZE LIVE RECORDING |
| **[STEP EDIT]** (in normal playback) | Enter STEP EDIT mode |
| **[PLAY]** | Start sequencer playback |
| **[PLAY]** (while playing) | Pause playback; press again to resume |
| **[STOP]** | Stop playback |
| **[STOP] + [STOP]** (quick double-press) | Stop playback with a short fade out of all tracks and send effects |
| **Hold [PAGE] + [LEFT] / [RIGHT]** | Change sequencer page (in GRID RECORDING) |
| **Hold [PAGE] + [TRIG]** | Jump to that page (in GRID RECORDING) |
| **[PAGE]** | Select the active pattern page (if more than 16 steps) |
| **[FUNC] + [TRIG]** (in GRID RECORDING) | Add a lock trig |

## MUTES

| Shortcut | Function |
|----------|----------|
| **[MUTE]** | Enter MUTE mode |
| **[MUTE] + [TRIG] / [SUBTRACKS]** | Quick mute/unmute one or several tracks |
| **[MUTE] + [UP]** | Enter GLOBAL MUTE mode |
| **[MUTE] + [DOWN]** | Enter PATTERN MUTE mode |
| **[MUTE] + [UP] / [DOWN]** (in MUTE mode) | Switch between GLOBAL and PATTERN MUTE modes |

## PARAMETER EDITING

| Shortcut | Function |
|----------|----------|
| **[PARAMETER PAGE] key + [PLAY]** | Reset all parameters on the selected parameter page to defaults |
| **[PARAMETER PAGE] key + [YES]** | Randomize parameters on that parameter page |
| ***DATA ENTRY* knob + [NO]** | Reset the individual parameter to its default value |
| **Press and turn *DATA ENTRY* knob** | Adjust parameter in larger increments |
| **Hold [FUNC] while turning *DATA ENTRY*** | Snap parameter values to musically relevant values (octaves, center, etc.) |
| **Press and hold [PARAMETER PAGE]** | See the values for all parameters on that page |

## SONG MODE

| Shortcut | Function |
|----------|----------|
| **[FUNC] + [SONG]** | Toggle SONG mode on/off |
| **[SONG] + [TRIG 1–16]** | Select a song and enter SONG mode |
| **[SONG] + [RIGHT]** | Open the SONG EDIT screen |
| **[SONG] + [LEFT]** (in SONG mode) | Loop the currently playing row; press again to stop looping |
| **[SONG] + [UP] / [DOWN]** (in SONG mode) | Select a specific song row to jump to and play next |
| **[STOP] + [STOP]** (in SONG mode) | Move the playhead to the song's beginning |

### Song Edit Screen

| Shortcut | Function |
|----------|----------|
| **[FUNC] + [RECORD]** | Copy the selected row |
| **[FUNC] + [STOP]** | Paste a previously copied row to the selected row |
| **[FUNC] + [PLAY]** | Reset the selected row's settings (default BPM, length, mute) |
| **[FUNC] + [DOWN]** | Add a new row below the selected row (copy of selected) |
| **[FUNC] + [UP]** | Delete the selected row |

## MISCELLANEOUS

| Shortcut | Function |
|----------|----------|
| **[FUNC] + [UP/DOWN]** (in quick scrolling menus) | Move the cursor one menu page at a time |
| **[YES]** | Confirm, select, enter sub-menus, tick/untick boxes |
| **[NO]** | Cancel, deselect, go back one step, perform Undo action |
| **[FUNC] + [NO]** (secondary) | Undo action |
| **[FUNC] + [YES]** (secondary) | Redo action |
| **Hold POWER then press [YES]** (or press POWER twice) | Turn off the device |
| **Hold POWER button** (unresponsive device) | Force power off |
| **[FUNC] + [TRIG]** (on NAMING pop-up) | Switch character layouts |
